Summary

Installs, configures, and manages the CUPS service.

Overview

All resources in this module require the CUPS daemon to be installed and configured in a certain way. To ensure these preconditions you should include the main cups class wherever you use this module.

Examples:

Install and run the CUPS server using default values

include '::cups'

Additionally set the default printer

class { '::cups':
  default_queue => 'Office'
}

Install HP PPD files before managing any printer queue

package { 'hplip':
  ensure => present
}

class { '::cups':
  require => Package['hplip']
}

Parameters:

  • default_queue (Optional[String]) (defaults to: undef) —

    The name of the default destination for all print jobs. Requires the catalog to contain a cups_queue resource with the same name.

  • listen (Variant[String, Array[String]]) (defaults to: ['localhost:631', '/var/run/cups/cups.sock']) —

    Which addresses to the CUPS daemon should listen to. Accepts (an array of) strings. Note that the cupsd.conf directive Port 631 is equivalent to Listen *:631. Warning: For this module to work, it is mandatory that CUPS is listening on localhost:631.

  • package_ensure (String) (defaults to: 'present') —

    Whether CUPS packages should be present or absent.

  • package_manage (Boolean) (defaults to: true) —

    Whether to manage package installation at all.

  • package_names (Variant[String, Array[String]]) (defaults to: $::cups::params::package_names) —

    A name or an array of names of all packages needed to be installed in order to run CUPS and provide ipptool. OS dependent defaults apply.

  • papersize (Optional[String]) (defaults to: undef) —

    Sets the system's default /etc/papersize. See man papersize for supported values.

  • purge_unmanaged_queues (Boolean) (defaults to: false) —

    Setting true will remove all queues from the node which do not match a cups_queue resource in the current catalog.

  • resources (Optional[Hash]) (defaults to: undef) —

    This attribute is intended for use with Hiera or any other ENC.

  • service_enable (Boolean) (defaults to: true) —

    Whether the CUPS services should be enabled to run at boot.

  • service_ensure (String) (defaults to: 'running') —

    Whether the CUPS services should be running or stopped.

  • service_manage (Boolean) (defaults to: true) —

    Whether to manage services at all.

  • service_names (Variant[String, Array[String]]) (defaults to: 'cups') —

    A name or an array of names of all CUPS services to be managed.

  • web_interface (Optional[Boolean]) (defaults to: undef) —

    Boolean value to enable or disable the server's web interface.
