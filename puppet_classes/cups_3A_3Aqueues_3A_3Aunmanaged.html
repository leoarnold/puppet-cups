Summary

A convenience wrapper around the `resources` metatype.

Overview

Private class

This class inherits the purge_unmanaged_queues attribute from the public puppet_classes::cups class. If set to true all CUPS queues not managed in the current Puppet catalog will be deleted.

Examples:

This class is implicitly declared through

class { '::cups':
  purge_unmanaged_queues => true
}
